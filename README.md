# MiniTwitter
A twitter like api with minimal functions that supports offline browsing and offline creation of tweets.

#Frameworks used:

###For Twitter APIs
* Fabric
* TwitterCore
* TwitterKit

###For offline image cache
* HanekeSwift

###For HUD display
* JGProgressHUD

###For database
* RealmSwift

###For parsing json objects
* SwiftyJSON
